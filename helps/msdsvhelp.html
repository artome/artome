dsv2db (Delimited Separated Values to mysql tables) is the program that is going to parse the instrument rawdata to filll datababe tables.


So creating a new instrument is not only creating suitable tables to store data but also provide sufficient information to a valid raw data parsing.


It is mandatory to put in the table name widget a semi-colon splited string with the table name and a short instrument description:

eg: CIMS;Chemical Ionization Mass Spectrometer.

The tablename should be a valid MYSQL name it must start by a letter and then besides alphanumeric chars can have underscore char _.
